# Python Course

Este repositorio contiene un curso de Python pensado para aprender desde cero, avanzando paso a paso desde los fundamentos del lenguaje hasta el desarrollo de un proyecto final con visualizacion e interactividad.

## Que aprenderas en este curso

A lo largo de 8 clases veremos los conceptos esenciales de Python, la logica de programacion y herramientas practicas para trabajar con datos, crear graficas y estructurar mejor el codigo.

## Clases del curso

1. **Primeros pasos con Python**  
   Introduccion al entorno de trabajo, sintaxis basica y primeros programas.

2. **Datos, operaciones y decisiones**  
   Tipos de datos, operadores y estructuras condicionales para tomar decisiones.

3. **Ciclos y repeticion**  
   Uso de bucles para automatizar tareas y repetir procesos de forma eficiente.

4. **Listas y manejo de datos**  
   Creacion, acceso y manipulacion de listas para organizar informacion.

5. **Funciones y organizacion del codigo**  
   Definicion de funciones y buenas practicas para escribir codigo reutilizable.

6. **Graficas y datos visuales**  
   Introduccion a la visualizacion de datos mediante graficas sencillas.

7. **Tablas y analisis de datos**  
   Trabajo con datos tabulares para explorar, ordenar y analizar informacion.

8. **Interactividad y proyecto final**  
   Integracion de lo aprendido en un proyecto practico con elementos interactivos.

Brindar una ruta clara y progresiva para que cualquier persona pueda iniciar en Python, practicar con ejemplos concretos y construir una base solida en programacion.
